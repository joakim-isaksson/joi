# Joi UI Events

Scriptable Object based UI Events for Unity

## Installation

```json
{
    "dependencies": {
        "com.joi.unityevents": "https://github.com/joakim-isaksson/joi.git#joi-unityevents",
        "com.joi.uievents": "https://github.com/joakim-isaksson/joi.git#joi-uievents"
    }
}
```
